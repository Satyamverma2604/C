# C Programming Quiz

Welcome to the C Programming Quiz! Test your knowledge with these questions and check your answers at the end.

---

1. What will be the output of the following code snippet?

int main() {
    int a = 3, b = 5;
    int t = a;
    a = b;
    b = t;
    printf("%d %d", a, b);
    return 0;
}

A) 35  
B) 53  
C) 33  
D) 55
